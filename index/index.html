1.   What is JavaScript?

 JavaScript, often abbreviated JS, is a programming language that is one of the core technologies
of the World Wide Web, alongside HTML and CSS. Over 97% of websites use JavaScript on the client side for web page behavior, often incorporating third-party libraries.

  bofore being run.In the content of a Wed browser,Scripting usually refers to
  program code written in javascript that is executed by the browser When a page is loaded,
